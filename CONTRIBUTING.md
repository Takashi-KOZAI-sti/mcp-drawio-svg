# Contributing

## Setup

```bash
git clone https://github.com/tkozai/mcp-drawio-svg.git
cd mcp-drawio-svg
npm install
npm run build
```

## Development

```bash
npm run dev   # Run with tsx (no build step)
npm run build # Compile TypeScript → dist/
```

## Submitting a Pull Request

1. Fork the repository and create a branch from `main`.
2. Make your changes.
3. Run `npm run build` and confirm there are no TypeScript errors.
4. Open a pull request with a clear description of the change.

## Reporting Issues

Please use [GitHub Issues](https://github.com/tkozai/mcp-drawio-svg/issues) to report bugs or request features.
